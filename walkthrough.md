# Ankush Kumar's Premium Portfolio Walkthrough

This project is a high-end, responsive portfolio website designed specifically for a B.Tech student showcasing a blend of technical prowess and modern design aesthetics.

## 🚀 Features

-   **Premium Dark Glassmorphism Design**: Using a curated palette of deep blues and purples with high-contrast UI elements.
-   **Interactive Tech Stack**: Section for technical skills (Frontend, Backend, etc.) with custom hover effects.
-   **Featured Projects Showcase**: Advanced card design with hover overlays and links for Github/External sites.
-   **SEO Optimized**: Semantic HTML5 structure with proper meta tags for search visibility.
-   **Modern Typography**: Using Google Fonts 'Outfit' for headings and 'Inter' for body readability.
-   **Micro-animations**: Smooth Reveal-on-Scroll animations, cursor glow follow effect, and animated hero-blob image.

## 🛠️ How to Customize

### 1. Update Your Profile Photo
The current photo is a generated placeholder. To use your actual photo (the one you provided), follow these steps:
- Rename your photo to `profile.png` (or `profile.jpg`).
- Replace the existing `profile.png` in the `c:\Users\ankus\portfoli\` folder with your file.
- Update the `<img>` tag in `index.html` if the extension is different.

### 2. Update Personal Details
Open `index.html` and search for:
- `ankush@example.com` - Update with your actual email.
- `social-links` - Update with your LinkedIn, Github, and Twitter URLs.
- `Project Title` - Change the project descriptions and images to reflect your real work!

### 3. Add More Skills
In `index.html`, you can find the `skills` section. Simply duplicate a `<div class="skill-card">` block and change the icon and text to add new skills.

## 🌐 Viewing the Site
To view your portfolio, simply open `index.html` in any modern browser.
